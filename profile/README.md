## 🧭 About X-THON

**X-THON**은 오픈 소스 기반의 융합·창의 해커톤입니다.
개발자, 디자이너, 기획자 등 다양한 전공의 학생들이 한 팀을 이루어 AI 전환(AX) 시대에 청년의 성장을 지원하는 디지털 혁신 문제를 해결하고, 혁신적인 아이디어를 실현합니다.

- 💡 **목표:** AI 전환(AX) 시대에 청년의 성장을 지원하는 디지털 혁신를 해결하는 창의적·혁신적 프로젝트 개발
- 🌍 **형태:** 완전 공개형 해커톤 (모든 저장소는 Public)  
- 🧑‍🤝‍🧑 **참가자:** 동국대학교 재학생 (전공무관) 
- 🏗️ **진행:** GitHub Organization 단위의 팀 운영

## 🧩 How to Participate

1. **GitHub Organization 생성**
   - 팀 이름으로 Organization을 만들고 모든 Repository를 Public으로 설정하세요.
   - 예시: `X-THON-2025-Team0_Nexus`

2. **팀 리드미 작성**
   - 팀 소개, 아이디어 개요, 역할 분담, 개발 계획 등을 `README.md`에 정리합니다.

---

## 📜 Rules

- 모든 코드는 **공개(Public)** 로 유지해야합니다.
- 외부 오픈소스 사용 시 **라이선스 명시** 필수.  
- 프로젝트는 팀 구성원 전원의 **기여 내역(Commit 기록)** 을 포함해야 합니다.  
- 제출물은 다음을 포함해야 합니다:  
  - 프로젝트 개요 한 줄 소개 
  - 완성된 조직 github 링크
