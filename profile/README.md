## 🧭 X-thon 2025란?

> **X-thon 2025**는 개발자, 디자이너, 기획자들이 모여  
> 창의적이고 혁신적인 프로젝트를 함께 만드는 오픈 해커톤 조직입니다.  

모든 저장소(Repository)는 **Public(공개)** 상태로 유지되며,  
각 팀은 동일한 규칙 하에서 독립적으로 자신의 프로젝트를 운영합니다.  
팀원들은 자유롭게 협업하고, 다른 팀의 결과물도 참고할 수 있습니다.

## 📜 협업 규칙

| 규칙 | 내용 |
|------|------|
| **Public-only 원칙** | 모든 저장소는 공개(Public) 상태로 유지해야 합니다. |
| **이름 규칙** | `2025_X_Thon_TeamN_ProjectName` 형태로 작성 (한글, 공백 ❌) |
| **팀장 권한** | Maintainer(팀장)는 팀원 관리 및 Repo 설정 권한을 가짐 |
| **PR / 브랜치 정책** | 각 팀 내부 자율 결정 (Owner는 개입하지 않음) |
| **삭제 / 이관 제한** | 저장소 삭제 및 이관은 Owner 승인 후 가능 |

## 🚀 팀별 시작 가이드

1️⃣ **팀장(Maintainer)** 은 `X-thon-2025` 조직 내에서 새 저장소를 생성  
 → 반드시 **Owner: X-thon-2025**, **공개(Public)** 로 설정  

2️⃣ **팀 연결:**  
 → `Organization → Teams → [내 팀] → Repositories → Add Repository`  

3️⃣ **README 작성:**  
 → [📄 팀 README 템플릿](https://github.com/X-thon-2025/guidelines/blob/main/team_readme_template.md) 참고  

4️⃣ **협업 시작:**  
 → 팀원 초대 후, 자유롭게 브랜치/PR 전략을 설정하여 개발 진행 🚀
